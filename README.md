# Local News CMS Platform
## Academic Project | Semester 3

### Overview
This project was developed as part of a Semester 3 college group assignment. The goal was to design a simplified content management system (CMS) for a local news platform where users could view, publish, and manage news content.
The project focuses on understanding CRUD operations, basic authentication logic, and responsive web design rather than production-level backend infrastructure.

### Team Project
Developed by a team of four  
This repository showcases the academic implementation of the project

### My Contribution
Implemented PHP logic for content handling  
Built add / edit / delete functionality for news posts  
Worked on a significant portion of the frontend UI  
Managed JSON-based storage for users and content  

### Features
User signup and login (JSON-based authentication)  
Publish, edit, and delete news articles  
Content stored and managed using JSON files   
Responsive UI built with Bootstrap  
Local development using Live Server  

### Tech Stack
PHP  
HTML, CSS, JavaScript  
Bootstrap  
JSON (for users and content storage)
